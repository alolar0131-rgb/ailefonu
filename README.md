# 🏛️ Aile Fonu – Resmi Web Sitesi

> **"Birlikte daha da güçlü geleceğe…"**

Aile Fonu, Haziran 2023'te kurulan; dayanışma, güven ve şeffaflık ilkeleri üzerine inşa edilmiş bir aile yardımlaşma yapısıdır.

---

## 🌐 Site Hakkında

Bu web sitesi, Aile Fonu'nun:
- Kuruluş hikayesini
- Misyon ve vizyonunu
- 2023–2026 yılları arasındaki tüm faaliyetlerini
- Yönetim kurulu yapısını
- Temel kurallarını

şeffaf bir şekilde tüm aile üyelerine sunmak amacıyla oluşturulmuştur.

## 🚀 GitHub Pages'te Yayınlama

1. Bu repoyu GitHub'a yükleyin
2. **Settings → Pages → Source: main branch / root** seçin
3. Birkaç dakika içinde siteniz `https://kullanici-adi.github.io/repo-adi` adresinde yayında olur!
